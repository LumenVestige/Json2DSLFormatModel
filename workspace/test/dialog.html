模型说明

模型简介：

  • 本模型为基于 T5 自训练专项模型，在原始 T5 模型架构上进行了专用领域数据的深度自训练。
  • 模型能够高效解析复杂层次的 JSON 数据结构，优化了在列表页、落地页等 JSON 格式中的特定字段识别与结构化数据推理的能力。
  • 提升了对目标领域的精准性和泛化表现。

使用说明：

  • 试试手气: 随机抽取一条从未参与训练的数据样本（列表页或落地页 JSON 数据）进行模型推理测试。
  • 开始推理: 使用 T5 自训练专项模型执行推理输出UI DSL JSON数据。

支持标签：

  • 布局类型：block，flex，list，li_item，card
  • 文本类型：title，paragraph，text，bold_text，small_text，italic_text，address
  • 资源类型：img，video，media，avatar
  • 链接类型：link

支持属性：

  • 布局属性：flex-justify-content，flex-align-items
  • 内容属性：content，src，href
  • 逻辑属性：for，if，else-if，else
  • 其他属性：border-radius
